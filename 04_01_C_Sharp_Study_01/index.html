第一节、安装C#及开发环境

第二节、开始C#之旅

  2.1 Hello World
  C#程序基本结构

2.1、命名空间

C#程序中的一种代码组织形式，主要用来标识类的可见范围，即命名空间必须加上类。
语法：namespace+命名空间
引用：using+命名空间;，切记 ; 不能丢。
PS：将自动生成与项目名称相同的命名空间。
举例：当未引用Test时，报错，如下图

然后将Demo引用，即错误消除：
using Demo;

2.2、类

一种数据结构，存储数据成员、方法成员和其它类等内容，便于调用。
语法：

class  类名
{
      // 类中的代码
}

PS：a）一个命名空间中可以有许多类；b）同一个命名空间中两个类的名称不能相同。

2.3、Main方法：程序的启动器

C#程序可以分为控制台程序和窗体程序，即CMD命令和应用软件，由Main来确定它的类型。
基本要求：
1）、Main方法必须定义为static;
2）、Main的首字母必须大写；
3）、返回值只有两种，void和int；
4）、命令行参数可选可不选，即Main后面括号内的内容可以删除，删除后程序可正常运行；
PS：使用void时，不会有返回值；使用int时，需要在代码中必须添加return+数字;。
语法：

static void Main ( string[ ] args )  {  }
static void Main ( ) {  }
static int  Main ( string[ ] args )  {  }
static int  Main ( ) {  }

常见错误提示1：

原因：1）没有Main方法；2）Main小写成main；3）没有使用static；
常见错误提示2：

原因：Main方法的返回值不是void或者int。
常见错误提示3：

原因：项目有多个入口，即有多个Main方法，需要在项目-属性-应用程序-启动对象中，设置默认的启动对象。

2.4、标识符

即名字，但是取名需要规范，具体如下：
首字母：1）字母；2）下划线‘ _ ’；3）首字母不能使用数字开头；
其余部分：任意多的：1）数字；2）字母；3）下划线‘_’；
注意事项：
1）标识符不能与C#关键字冲突，如int static = 30;是错误的；
2）标识符是区分大小写的，如int age = 30;与int Age = 30;是两个标识符;
3）标识符中不能使用百分号%。
C#常见关键词：程序中显示蓝色的字。
int public this finally boolean abstract
continue float long short throw return
break for foreach static new interface
if goto default byte do case
void try switch else catch private
double protected while char class using

2.5、C#中的语句

C#语句实质上就是实现各种功能的C#代码，最基本的程序构成单位。

2.6、注释：解释说明

1）行注释：
使用双斜杠//来表示，后面接注释的内容。同一行后续输入将不参与运行。
2）块注释：连续多行代码注释。

/*
 * 版权
 * 日期
 * 人员
 */

3）注释的作用：解释说明；不运行代码。
快捷按钮：工具栏，注释选中行，取消对选中行的注释；
快捷键：Ctrl+K,Ctrl+C；Ctrl+K,Ctrl+U。

2.7、程序编写规范

1）代码编写规则（死记硬背）：

  • 尽量使用接口编程；
  • 关键语句一定要编写注释；
  • 局部变量随用随声明；
  • 尽量少用goto语句；
  • 如果参数多，建议使用结构；
  • 避免对大段代码使用try...catch；
  • 同一个文件中避免编写多个类；
  • 字符串多变时，用StringBuilder；
  • if语句块中使用“{ }”；
  • switch语句中一定编写default；

2）常用的两种命名方法：

  • Pascal命名法：所有单词第一个字母大写，其它字母小写，如User、GetInfo。
  • Camel命名法（驼峰命名法）：除了第一个单词，所有单词第一个字母大写，其它字母小写，如userId，userName。

3）程序中各元素的命名规范：

  • 项目名：公司域名+项目名称，如：mingrisoft.ERP；
  • 命名空间名：公司名称/产品名称；
  • 接口：一切接口都用大写字母“I”开头；
  • 类名：功能与操作的完美结合；
  • 方法名：动宾关系，实现何种操作；
  • 成员变量：尽可能加前缀“ _ ”。

2.8、输出文字及窗口停留

//Write：输出后不会换行，光标定位在同一行的最后。
Console.Write("人因梦想而伟大");
Console.WriteLine("             ——雷军");
Console.ReadLine();//光标停留

代码显示：

//WriteLine：输出后会换行，光标定位到下一行的开始。
Console.WriteLine("人因梦想而伟大");
Console.WriteLine("             ——雷军");
Console.ReadLine();//光标停留

代码显示：

2、输出文字时光标停留
Console.ReadLine();//光标停留

同时输出多个变量：

Console.WriteLine(string.Format("{0} {4} {2} {2} {2}", money1, money2, money3, money4, money5));

变量与表达式 中提到System.Console.WriteLine({0}，{1},myString,myInteger);输出多个参数时，“每个占位符用一对{ }中加一个整数来表示。整数从0开始，每次递增1，占位符的总数应等于列表中指定的变量数，该列表用逗号分隔开，跟在字符串后面。”
实际上只要占位符的第1个整数为0就可以，其他整数小于变量总数就可以，不必每次递增1，也不必占位符的个数和变量的个数相等，但是占位符中的整数代表了变量列表的索引值。

第三节、变量与常量

3.1、变量的基本概念

概念：一个变量就是存储区（内存）中的一个存储单元。

3.2、变量的声明及赋值

1、使用变量的步骤

  • 声明一个变量：根据类型分配空间；
  • 初试化变量：将数据存入内存空间；
  • 使用变量：去除数据使用。

2、变量的声明
1）语法：
语法一：

  • 变量类型 变量名 = 数值 int money = 1000;

语法二：

  • 变量类型 变量名；int money;
  • 变量名 = 数值；money = 1000;

语法三：变量类型一致时，

  • 变量类型 变量名1,变量名2,...,变量名n; int money1,money2,...,moneyn;

2）变量的命名规则：
取名需要规范，具体如下：
首字母：1）字母；2）下划线‘ _ ’；3）首字母不能使用数字开头；
其余部分：任意多的：1）数字；2）字母；3）下划线‘_’；
注：

  • 首字母只能是字母或下划线" _ "。
  • 基本要求：简短且能清楚地表明变量的作用，通常第一个单词的首字母小写，其后单词的首字母大写。例如：myScore。
  • 变量不能与C#关键字冲突；
  • 变量严格区分大小写，即age和Age表示不一样的变量；
  • 变量里面不能带特殊符号（%，$）和空格。
  • 在C#语言中，允许使用汉字作为变量名，在程序运行时并不会出现什么错误，但建议读者尽量不要使用汉字作为变量名。

3）变量的初始化：使用变量前必须初始化——即赋值。
money = 368;在这里**=**为赋值运算符，将右边的数赋值给左边的变量。
使用方法：

  • 单独变量初始化

int sum;
sun = 368;

  • 声明时初始化变量

sun = 368;

  • 同时初始化多个变量

int a,b,c,d,e;
a = b = c = d = e = 0;

string s1, s2, s3;
s1 = s2 = s3 = "ling";

也可以使用已经赋值的变量对新的变量赋值。

string title2 = title;
Console.WriteLine("title2 = {0}",title2);

4）变量的作用域
类体：成员变量（全体变量），只要该方法在类中，就都可以使用；分为两种，静态变量和实例变量，主要区别是有没有关键词static。使用静态变量时，直接使用类的名称进行调用；若使用实例变量，则首先需要创立类的对象，然后再使用对象进行调用。

static int x = 90; //静态变量
int x = 45; //实例变量；

static void Main(string[] args)
  {
      //静态方法中调用静态变量
      Console.WriteLine(str2);
      //静态方法中调用实例变量
      Program p = new Program();
      Console.WriteLine(p.str);

方法体：局部变量，只能在该方法中使用，不能再其他方法体中使用。

//实例：输入并显示用户名
Console.WriteLine("Welcome to China!");
Console.WriteLine("Please input your username:");
String username = Console.ReadLine();
Console.WriteLine("The current user is "+username);

常见错误1：

String title;
Console.WriteLine(title);

报错：使用了未赋值的局部变量“title”。
纠错：String title = "CYU";

常见错误2：

int %hour = 18;
Console.WriteLine(%hour);

报错：无效的表达式项“%”。
纠错：int hour = 18;，变量的命名不能以特殊符号开始。

常见错误3：

String name = "san";
String name = "si";

报错：已在此范围定义了名为“name”的局部变量。
纠错：String Name = "si";，变量的命名不能重复。

3、变量的类型：数据类型

  • 值类型：
    • 简单：整数、实数、字符、布尔；
    • 复合：结构，枚举；
  • 引用类型：类、接口、数组、委托；

1）值类型：直接储存值，对一个值类型的改变，并不会影响到另外一个值类型；
例：

int age = 30;
int age2 = age;
age2 = 35;
Console.WriteLine(string.Format("{0} {1}", "age=" + age, "age2=" + age2));
Console.WriteLine("age2=" + age2);

输出为：

age=30 age2=35
age2=35
  • 简单类型：数值型（int/float），字符型（Char），布尔型（bool）。

整数类型：

类型 说明（8位等于1字节） 范围
sbyte 8位有符号整数 -128~127
short 16位有符号整数 -32768~32767
int 32位有符号整数 -2147483648~2147483647
long 64位有符号整数 -9223372036854775808~9223372036854775807
byte 8位无符号整数 0~255
ushort 16位无符号整数 0~65535
uint 32位无符号整数 0~4294967295
ulong 64位无符号整数 0~18446744073709551615

PS：前四种可以储存负数，而后四种无符号整数不能储存负数，如unit n = -10;，是错误的。
